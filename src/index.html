Additional Information

There is additional data for each panel that is displayed when a spin is complete and the prize displayed, and when the list of prizes for a group is displayed.

Spinning the Wheel

Clicking on the left or right side of the screen will cause the wheel to spin counter-clockwise or clockwise respectively. Alternatively, the left and right keys will do the same.

Quick Change

Panels and people can be quickly removed from the wheel and group list respectively with the checkboxes on the main menu.

Group Spinning

There is an option for a "group" to be defined and a person selected for each spin. When this is done, the information is recorded and can be displayed when spinning is done. Spinning needs to occur for each individual listed

The Code

The code within this project is not minified or mangled to allow it to be examined at any point.
